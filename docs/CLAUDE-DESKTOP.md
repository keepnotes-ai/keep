# Using keep in Claude Desktop

Install the keep extension for Claude Desktop:

```
keep config mcpb
```

This generates and opens a `.mcpb` bundle that registers keep's MCP server with Claude Desktop. All keep tools become available immediately.

## Getting started

Once the extension is installed, ask Claude:

> "Please read all the `keep_help` documentation, and then use `keep_now`."

This gives Claude the full context of how keep works, and starts your first reflective memory session.
