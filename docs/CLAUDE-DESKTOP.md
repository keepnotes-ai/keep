# Using keep in Claude Desktop

Install the keep extension for Claude Desktop:

```
keep config mcpb
```

This generates and opens a `.mcpb` bundle that registers keep's MCP server with Claude Desktop. All keep tools become available immediately.

## Getting started

Once the extension is installed, just say to Claude:

> Please read all the `keep_help` documentation, and then use `keep_prompt(name="reflect")`
