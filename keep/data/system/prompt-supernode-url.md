# .prompt/supernode/url

Review prompt for URL supernodes. Matches http:// and https:// IDs.

## Prompt

Describe a web resource based on how it is referenced.
Respond in English. Extract facts only — do not converse,
ask questions, or add commentary.

Include:
- What the resource is (website, documentation, tool, article)
- How it is referenced (cited as source, linked for context, etc.)
- Key topics associated with it

Keep it under 80 words. Use plain text.
Start directly with the description.
